# GitHub Action: Daily Google Sheet Updates (7 PM Dubai Time)

## Overview
Setup a GitHub Action that runs daily at 7 PM Dubai time (Asia/Dubai, UTC+4) to execute all 6 screening strategies and append results to a Google Sheet with one sheet per strategy.

## Required Permissions & Setup

### 1. Google Cloud Service Account (CRITICAL)
Create a Google Cloud Service Account with these permissions:

**Google Cloud Console Steps:**
1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project or select existing one
3. Enable Google Sheets API:
   - Navigation Menu → APIs & Services → Library
   - Search "Google Sheets API" → Enable
4. Create Service Account:
   - IAM & Admin → Service Accounts → Create Service Account
   - Name: `github-actions-sheets` (or similar)
5. Generate JSON Key:
   - Click service account → Keys tab → Add Key → Create JSON key
   - **Download and save securely** - this contains the private key

**Service Account Email Format:** `github-actions-sheets@your-project.iam.gserviceaccount.com`

### 2. Google Sheet Sharing
1. Open your empty Google Sheet
2. Click Share button
3. Add the service account email (from step 5 above) as Editor
4. Copy the Google Sheet ID from URL:
   - URL: `https://docs.google.com/spreadsheets/d/{SHEET_ID}/edit`
   - The `SHEET_ID` is between `/d/` and `/edit`

### 3. GitHub Repository Secrets
Add these secrets in GitHub repo → Settings → Secrets and variables → Actions:

| Secret Name | Value |
|-------------|-------|
| `GOOGLE_SHEETS_CREDENTIALS` | Full JSON content of the service account key file |
| `GOOGLE_SHEET_ID` | The spreadsheet ID from step 2 above |

## Timezone Reference
- **Dubai Time:** Asia/Dubai (UTC+4, no DST)
- **7 PM Dubai** = 15:00 UTC
- GitHub Actions cron format: `0 15 * * *` (runs at 15:00 UTC daily)

## Dependencies
The workflow needs these Python packages (add to existing requirements or install via pip):
- `gspread` - Google Sheets API client
- `google-auth` - Authentication library

## Error Handling
- If no stocks pass all filters, the scripts write "No picks/No winner" row
- This row should still be appended with date for historical tracking
- GitHub Action should continue even if individual strategies fail

## Cost Considerations
- Google Sheets API has generous free tier (500 requests/100 seconds per user)
- Daily run = 6 requests (one per strategy)
- Well within free tier limits
